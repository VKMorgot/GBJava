Крестики-нолики в процедурном стиле

1. Создание игрового поля
1.1 Размер поля - константа
    - 3х3
    - 5х5
1.3 Количество фишек для победы - константа

2. Ячейки поля
2.1 Символы 'X', 'O', '•' - константы

3. Инициализация поля - метод initMap()
Инициализируем поле заданного размера и заполняем его пустыми значениями

4. Вывод поля в консоль - метод printMap()
Создаем шапку. По сути, систему координат для поля

5. Ход человека - метод humanTurn()
Делаем через Scanner -> Ввод координат в формате X Y
Предусмотреть удобство ввода, т.е. начало координат не от 0, а от 1.

6. Проверка ячеек - метод isCellValid(x, y)
Проверяем возможность установки фишки в ячейку

7. Ход компьютера - метод aiTurn()
Сделан через Random

8. Проверка победы - метод checkWin(symb)
Сделан через перебор всех возможных значений победы

9. Проверка, пустое поле или нет - метод isMapFull()

10. Игра - метод main()
Инициализируем поле
Выводим поле
Пока возможно играем:
Делаем ход -> печатаем результат -> проверяем победу
Проверяем, есть ли место для хода
Передаем ход
